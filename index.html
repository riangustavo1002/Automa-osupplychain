02 — A Solução

Google Apps Script:
lógica que automatiza o trabalho

GAS · JavaScript consolidarApenasAbasNumericas.gs 
// REGRA DE OURO: só consolida abas cujo nome contém números
// Ex: "13600" ✓ | "123 - Forn" ✓ | "RESULTADO" ✗

function consolidarApenasAbasNumericas() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const abas = ss.getSheets();
  const abaDestinoNome = "BASE_CONSOLIDADA_FINAL";
  let abaDestino = ss.getSheetByName(abaDestinoNome) || ss.insertSheet(abaDestinoNome);

  // Cabeçalho normalizado
  const cabecalho = [["Cód Fornecedor", "Nome Fornecedor", "Cód Item",
                       "Descrição", "Período (Mês)", "Qtd Comprada", "Valor Comprado"]];

  const meses = ["2025-01", "2025-02", ... , "2025-12"];
  let dadosFinais = [];

  abas.forEach(aba => {
    // Filtra somente abas com código numérico (fornecedor)
    if (/\d/.test(aba.getName())) {

      let dados = aba.getDataRange().getValues();

      // Percorre linhas que começam com número (evita cabeçalhos)
      dados.forEach(linha => {
        if (/^\d/.test(linha[0])) {

          // Transpõe colunas horizontais de mês → linhas verticais
          for (let col = 4; col <= 27; col += 2) {
            const qtd = linha[col];
            const valor = linha[col + 1];

            // Só grava se houver movimentação real
            if (valor > 0 || qtd > 0) {
              dadosFinais.push([codForn, nomeForn, codItem,
                               descItem, meses[mesIdx], qtd, valor]);
            }
          }
        }
      });
    }
  });

  // Escrita em lote — performance máxima
  abaDestino.getRange(2, 1, dadosFinais.length, 7).setValues(dadosFinais);
}
🔍

Regex de Filtro de Abas

/\d/.test(nome) — identifica abas de fornecedor pelo código numérico, ignorando abas auxiliares como "RESULTADO" e "PARÂMETROS".

↕️

Transposição Horizontal → Vertical

Converte o layout de colunas por mês (Qtd Jan | Vlr Jan | Qtd Fev...) em registros normalizados, um por período — viabilizando análise temporal.

⚡

Escrita em Lote (Batch Write)

Toda a consolidação é acumulada em array e escrita de uma vez com setValues() — evitando chamadas repetidas à API e garantindo performance.

🔗

Cruzamento com PROCX + SOMASES

Base consolidada enriquecida com dados de outras planilhas via fórmulas avançadas — preço médio, categoria, localização e totais por período.
